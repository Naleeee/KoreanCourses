# Chapter 6

## Time

### Structure

To give an hour in Korean you must use this structure:

[Hours with classic alphabet] 시 [Minutes with sino alphabet] 분

샤 means hour and 분 means minutes.

### Examples

| Time (english)     | Time (Korean) |
| ------------------ | ------------- |
| 4:20               | 네시:이분     |
| 8:35               | 여섯:삼오분   |
| 10:55              | 열시:오십오분 |
| 00:10 (said 12:10) | 열두시:십분   |
